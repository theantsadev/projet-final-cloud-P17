Avec Android Studio:
- Gradle est déjà installé dans Android Studio
- Pas besoin de télécharger de dépendances
- Vous pouvez déployer directement sur votre smartphone

1.Lancez Android Studio

2.Ouvrez le projet :
    File → Open
    Naviguez vers : android
    Cliquez sur OK

3.Attendez la synchronisation Gradle (première fois uniquement)

4.Préparez votre smartphone :
    Paramètres → À propos → Appuyez 7 fois sur "Numéro de build"
    Paramètres → Options développeur → Activez "Débogage USB"
    Connectez via USB au PC
    Acceptez l'autorisation sur le téléphone

5.Déployez l'app :
    Dans Android Studio, cliquez sur ▶️ Run 'app'
    Sélectionnez votre smartphone
    L'app s'installe et démarre automatiquement
